Calcolatrice in C++

Dettagli del Progetto

Questa calcolatrice da console è stata sviluppata in C++ come parte del mio percorso di apprendimento della programmazione. L'applicazione offre sia funzionalità di calcolo aritmetico di base che funzioni geometriche avanzate, permettendo all'utente di effettuare diversi tipi di calcoli in un'unica interfaccia.

< in collaborazione con Buscema >


Funzionalità principali:

  • Operazioni aritmetiche di base (addizione, sottrazione, moltiplicazione, divisione)
  • Calcoli di potenze e radici quadrate
  • Funzioni trigonometriche (seno, coseno, tangente)
  • Funzioni trigonometriche inverse

Tecnologie utilizzate:

C++ Console Application Standard Libraries
